# Mynet Local - İstanbul Haber Portalı

Flask + Google News RSS kullanarak İstanbul geneli ve ilçe bazlı haberleri listeleyen yerel haber portalı.

## Özellikler

- Son dakika (sadece İstanbul)
- İlçe bazlı haberler
- Otomatik stok görsel (haber başlığına göre)
- Reklam alanları (üst / sağ / sol / grid içi)

## Kurulum

pip install -r requirements.txt  
python app.py

Tarayıcı: http://127.0.0.1:5000
